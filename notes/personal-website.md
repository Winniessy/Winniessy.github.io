# 个人网站搭建记录

日期：2026-07-13  
标签：个人网站 / Docsify / Markdown

## 目标

搭建一个简单、容易维护的个人网站，用来展示项目和记录学习过程。

## 当前方案

使用 Docsify：

- `index.html` 负责加载 Docsify。
- `README.md` 作为首页内容。
- `_sidebar.md` 作为导航。
- `notes/*.md` 保存每篇学习记录。

## 为什么这样做

这个方案不用构建工具，不需要 npm，也不需要复杂主题。以后主要写 Markdown，就能更新网站。
